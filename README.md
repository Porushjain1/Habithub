# HabitHub 🎯

Welcome to **HabitHub**, a premium, aesthetic, and fully-featured habit tracking application designed to help you build consistency, track your goals, and visualize your progress like never before.

## ✨ Features

- 📊 **Spreadsheet Tracker**: A lightning-fast, Google Sheets-inspired daily grid for checking off your habits.
- 🐙 **GitHub-Style Heatmap**: Visualize your yearly consistency with a beautiful contribution graph that mimics your GitHub profile.
- 📅 **Weekly Calendar Planner**: Plan and execute your days with a specialized column-based weekly planner layout.
- 📈 **Deep Analytics**: Dynamic charts, consistency curves, and completion progress bars calculated automatically.
- 🏆 **Gamification & Streaks**: Level up as you hit "Perfect Days" and track your longest streaks seamlessly.
- 🌓 **Dark Mode**: A meticulously designed dark theme that activates automatically or via the top-bar toggle.
- 📱 **Fully Responsive**: Works perfectly on desktops, tablets, and mobile devices.

## 🛠 Tech Stack

- **Frontend Framework:** React 18 with Vite
- **Styling:** Tailwind CSS v4
- **State Management:** Zustand (with LocalStorage Persistence)
- **Charts:** Recharts
- **Icons:** Lucide React
- **Date Handling:** Date-fns

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/Porushjain1/Habithub.git
   ```
2. Navigate into the directory:
   ```bash
   cd Habithub
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser to `http://localhost:5173` to view the app!

## 🌍 Deployment
HabitHub is optimized for effortless deployment on platforms like Vercel and Netlify. 
Simply connect this repository to your Vercel account, leave the default build settings (`npm run build`), and hit deploy!
